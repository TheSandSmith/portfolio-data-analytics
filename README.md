## Project Overview

A comprehensive collection of SQL scripts for explorative analytics, and advanced analytical reporting. The source data is the business-ready layer (i.e. gold layer) of a data warehouse.

These scripts cover various exploratory analyses such as:

1. date exploration
1. measure exploration
1. magnitude analysis
1. ranking analysis

The scripts also cover various advanced analyses such as:

5. seasonality analysis
1. cumulative analysis
1. data segmentation
1. part-to-whole analysis
1. full reporting (_via complex views_)
